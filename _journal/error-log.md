## Легенда типов ошибок

- `error-handling` — некорректная обработка ошибок
- `naming` — нарушение соглашений об именовании
- `style` — нарушение Go style guide
- `logic` — логические ошибки в алгоритме
- `goroutine` — проблемы с горутинами (leak, race)
- `interface` — неправильное использование интерфейсов
- `performance` — неоптимальный код
- `testing` — проблемы с тестами
- `structure` — неправильная структура пакетов/проекта
