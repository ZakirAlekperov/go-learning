# Тема 1 — Синтаксис, типы, управляющие конструкции

## Go — что это такое?

Представь, что взяли C, убрали всё опасное (ручное управление памятью, указочная арифметика), добавили сборщик мусора как в Java, но сделали компиляцию молниеносной и выбросили ООП-церемонии.

Три главных принципа:
- **Простота** — в языке намеренно мало конструкций
- **Явность** — код должно быть легко читать, даже если видишь впервые
- **Производительность** — компилируется в нативный бинарник

## Переменные: var vs :=

```go
// Длинный способ — явное объявление
var x int = 5

// Короткий способ — вывод типа (только внутри функций)
x := 5
```

`:=` — оператор короткого объявления: одновременно объявляет переменную И присваивает значение. За пределами функций недоступен.

Go строго статически типизирован — как только тип определён, изменить нельзя. Неявных преобразований нет: `int32` и `int64` — разные типы.

## Константы

```go
const (
    StatusRunning = "running"
    StatusStopped = "stopped"
    StatusZombie  = "zombie"

    MemWarningThreshold = 200.0 // MB
)
```

Группировка через скобки — стандартный Go-стиль. Неиспользованные константы — не ошибка компиляции (в отличие от переменных).

## Массивы

Фиксированная длина, которая является частью типа. `[3]int` и `[5]int` — разные типы. Передаются по значению (копируются). В реальном коде используются редко.

```go
arr := [3]int{10, 20, 30}
arr := [...]int{10, 20, 30} // компилятор сам считает длину
fmt.Println(arr[0])         // 10
fmt.Println(len(arr))       // 3
```

## Слайсы

Слайс — дескриптор с тремя полями: указатель на массив, длина (`len`), ёмкость (`cap`). Это «окно над массивом».

```go
var s []int              // nil слайс, len=0, cap=0
s := []int{10, 20, 30}  // литерал
s := make([]int, 3)     // [0,0,0], len=3, cap=3
s := make([]int, 3, 10) // [0,0,0], len=3, cap=10

fmt.Println(len(s))
fmt.Println(cap(s))
```

### append

```go
s := []string{"nginx", "sshd"}
s = append(s, "mysql")          // добавить один
s = append(s, "redis", "php")   // добавить несколько
```

Результат ВСЕГДА присваивай обратно — после append указатель может измениться.

### Срезы (slicing)

```go
s := []int{0, 1, 2, 3, 4, 5}
s[1:4]  // [1, 2, 3]
s[:3]   // [0, 1, 2]
s[2:]   // [2, 3, 4, 5]
```

Срез не копирует данные — изменение элемента в срезе изменит оригинал.

## for range

```go
names := []string{"nginx", "sshd", "mysql"}

for i, name := range names { fmt.Println(i, name) } // индекс + значение
for i := range names { fmt.Println(i) }              // только индекс
for _, name := range names { fmt.Println(name) }     // только значение
```

`_` (blank identifier) — отброс ненужного значения. Объявленная, но неиспользуемая переменная — ошибка компиляции. `_` решает это.

## map

```go
counts := make(map[string]int)

counts["running"] = 5
counts["zombie"]++  // работает даже если ключа нет — вернёт 0 и прибавит 1

fmt.Println(counts["running"]) // 5

for status, count := range counts {
    fmt.Printf("%-10s : %d\n", status, count)
}
```

Обращение к несуществующему ключу не паникует — возвращает нулевое значение типа.

## Поиск максимума

```go
memory := []float64{45.2, 12.1, 210.5, 0.1}

maxMem := memory[0]
maxIdx := 0

for i, m := range memory {
    if m > maxMem {
        maxMem = m
        maxIdx = i
    }
}
```

## switch

```go
switch status {
case "running":
    fmt.Println("✅ работает")
case "zombie":
    fmt.Println("☠️  зомби")
default:
    fmt.Println("неизвестно")
}

// switch без выражения — замена цепочке if/else
switch {
case mem > 500:
    fmt.Println("критическое потребление")
case mem > 100:
    fmt.Println("высокое")
default:
    fmt.Println("норма")
}
```

В отличие от Java — нет `break`. Каждый `case` заканчивается автоматически.

## fmt.Printf — форматированный вывод

```go
fmt.Printf("%-15s PID: %-6d MEM: %6.1f MB  [%s]\n", name, pid, mem, status)
```

| Глагол | Что делает |
|--------|------------|
| `%s`   | строка |
| `%d`   | целое число |
| `%f`   | float (`.2f` — два знака) |
| `%v`   | любое значение |
| `%T`   | тип значения |
| `%-10s`| выравнивание по левому краю, ширина 10 |
| `%6d`  | выравнивание по правому краю, ширина 6 |

`fmt.Sprintf` — то же самое, но возвращает строку:
```go
line := fmt.Sprintf("Процесс %s: %.1f MB", name, mem)
```
